# Desafio_WebAnalysis
Processo Seletivo de WebAnalysis

Stacks:
React
Vite
node.js
express
Gemini

Como rodar:

frontend:
npm i
npm run dev

backend (sem ia):
cd .\Desafio_WebAnalysis\Backend
npm i
npm i express cors @anthropic-ai/sdk dotenv
npm run dev

backend (com ia):
npm init -y
npm i express cors dotenv @google/generative-ai
npm run dev
